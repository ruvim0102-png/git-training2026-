git init — Сделать из обычной папки гитовую репозиторию. Первая команда, когда начинаешь новый проект.
git clone — Стянуть к себе на комп готовый проект вместе с историей.
git status — Посмотреть, какие файлы ты поменял, какие новые, а какие готовы к сохранению.
git add — Добавить конкретные файлы в список на сохранение.
git commit — Сохранить текущее состояние проекта (создать версию).
git branch — Посмотреть список веток или создать новую, чтобы не сломать основу.
git checkout — Переключиться на другую ветку или откатить файл до последней версии.
git push — Отправить свои сохраненные версии на Гитхаб, чтобы они там не потерялись.
